fireconsole.rep.js

This Guide is best viewed at fireconsole.github.io/fireconsole.rep.js using a Modern Browser.

A FireConsole component in the form of a jsonrep.github.io/jsonrep.

Integration

HTML Page

<div renderer="jsonrep">{
    "@./dist/fireconsole": {
        "messages": [
            "Hello World!"
        ]
    }
}</div>
<script src="https://jsonrep.github.io/jsonrep/dist/jsonrep.js"></script>
Result
{ "@./dist/fireconsole": { "messages": [ "Hello World!" ] } }

Development

Requirements:

  • bash 4
  • nvm - Node Version Manager

Build from source

rm -Rf dist/        # Optional to verify build
nvm use 9
npm install
npm run build       # Append '--ignore-dirty' if you removed 'dist/'
git diff            # Optional to verify build
    # No changes should be detected.
